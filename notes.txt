How to create jsp ? 
	right click on views folder ->  new -> other -> type -> jsp -> next -> give jsp name and finish 
	
	
How To create  controller ? 
	1) right click on controller folder -> new -> other -> type class -> next -> Give class name and finish 
	2) use @Controller annotation above the class name 
	
	
class =>
			1) variables 
			2) methods 

How to run project / start server ?
project -> right click -> run as -> spring boot application 



How to create entity ? 
	we have folder called entity inside grownited folder 
		if not then we have to create folder entity 
	now create class UserEntity inside entity folder 
	use @Entity and @Table(name="users") at class level annotation 
	add variable in your UserEntity like , userId, firstName etc. 
	
	
How to generate getters and setters? 
	in yout class do right click at blank space ,
	then click on source -> now click on generate getters and setters ->
	select all -> Generate.
